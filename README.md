# Crypto Tracker

Small CLI to fetch cryptocurrency prices using CoinGecko.

Usage

Install dependencies:

```bash
npm install
```

Get prices for defaults (bitcoin and ethereum):

```bash
npm start
```

Get prices for specific coins:

```bash
npm start -- bitcoin cardano
```

Run quick test:

```bash
npm test
```

Examples

Run the jsonplaceholder examples (POST and GET):

```bash
node examples/typicode.js
```
